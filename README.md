## A powerful website scanner

**Requirements:**
Install these Python packages:

**( pip install requests beautifulsoup4 rich tldextract python-whois certifi )**

## Optional API checks:

**( pip install  virustotal_python google-api-python-client )**

## Usage:

## 1.**Change directory to the file where you saved 'scanner.py' using 'cd' command on CMD (Command Prompt)**

## 2.**Type: 'python scanner.py {website}' DO NOT INCLUDE BRACKETS**
